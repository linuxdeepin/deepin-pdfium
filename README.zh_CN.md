# deepin-pdfium

Deepin上的pdf开发库。

### 编译依赖

- cmake,
- qt6-base-dev,
- pkg-config,
- libjpeg-dev,
- libicu-dev,
- libpng-dev,
- zlib1g-dev,
- liblcms2-dev,
- libopenjp2-7-dev,
- libfreetype6-dev,
- libgtest-dev,
- libchardet-dev

## 安装

### 构建过程

```
$ cd /path/to/your/source/code
$ mkdir build
$ cd build
$ qmake ../
$ make
$ sudo make install
```
