# deepin-pdfium

Deepin上的pdf开发库。

### 编译依赖

- qtbase5-dev
- chardet
-  lcms2 
- freetype2 
- libopenjp2

## 安装

### 构建过程

```
$ cd /path/to/your/source/code
$ mkdir build
$ cd build
$ qmake ../
$ make
$ sudo make install
```
